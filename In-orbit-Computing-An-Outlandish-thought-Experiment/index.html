In-orbit Computing An Outlandish thought Experiment?

Paper on scihub

  • In-orbit computing would be available from anywhere on the earth, unlike the terrestrial data centers
  • It will make it easier to implement an edge-computing solution without deploying a terrestrial infrastructure in many locations
  • Snowcone
  • It allows using unused orbit infrastructure to transport data and move part of our infrastructure to the orbit
  • Computing raw data from satellites, and sending them directly to the ground, will help limit the usage of satellite downlinks.
  • For many applications, like video chatting, we need the "meet-up" server with low latency. If we use in-orbit computing, we can pick a server especially for a set of users. It will minimize the latency of a connection.
  • The electric power would be the disadvantage
  • In-orbit compute services could be temporarily unavailable due to weather conditions
