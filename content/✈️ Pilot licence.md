# ✈️ Pilot licence
___

Licenses:
- PPL - Private pilot licence
- MPL
- CPL - Comercial pilot licence
- ATPL -  Airline transpot pilot licence
- SPL - gliders
- BPL - Balloon

* PPL(H) - helicopter
* PPL(AS) - airship
